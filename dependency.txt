Known bugs / limitations
- README currently describes the layout instead of including a real screenshot image.
- The app generates structured content and visual directions, but it does not yet create real layered images.
- Instagram direct publishing is not implemented yet.
- Duplicate checking uses text similarity, not semantic similarity.
- OpenAI generation still depends on API quota and billing; the new `Local Demo` mode is for testing, not production-quality AI writing.
